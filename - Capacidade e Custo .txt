Durabilidade e Disponibilidade
As mensagens publicadas no serviço de Streaming são replicadas de forma síncrona em três domínios de disponibilidade, quando disponíveis. Em regiões com um único domínio de disponibilidade, os dados são replicados em vários domínios de falha. Isso garante que mesmo a falha de um domínio de disponibilidade ou de falha não resulte em perda de dados. O resultado são dados altamente duráveis.
O Oracle Cloud Infrastructure fornece um acordo de nível de serviço (SLA) para Streaming . Consulte a página Contrato de Nível de Serviço do Oracle Cloud Infrastructure para obter detalhes.

Usando o Resource Manager para criar um stream
Resource Manager é um serviço Oracle Cloud Infrastructure ( OCI ) que permite automatizar o processo de provisionamento de seusrecursos OCI . Usando o Terraform, o Resource Manager ajuda você a instalar, configurar e gerenciar recursos por meio do modelo "infraestrutura como código".

Uma configuração do Terraform codifica sua infraestrutura em arquivos de configuração declarativos. A configuração define os recursos que você pretende provisionar, variáveis ​​e instruções específicas para provisionar os recursos

Você pode usar o Resource Manager ou a CLI do Terraform com o provedor OCI Terraform para provisionar recursos de streaming , como streams e pools de streams.
